# User Guide

Gigi is in a foundation rebuild.

The current runtime does not log in to Discord yet. The only live interface is the health/readiness HTTP surface used by local development, CI, and deploy checks.

## Current Live Checks

- `GET /healthz`: process is alive and reports build metadata.
- `GET /readyz`: process is configured and can reach PostgreSQL.

## Planned Discord Behavior

Future Discord slices are expected to restore and expand:

- DM conversation
- channel mentions
- structured commands
- plugin-provided commands such as music controls
- admin-installed plugin capabilities
- permission-aware task, relay, memory, and retrieval features

These are not current shipped behavior in this foundation slice.
